# Сайт-портфолио — Арина Некрасова

Одностраничный сайт-визитка детского психолога. Создан на основе дизайна из папки `Reference`.

## Запуск

```bash
npm install
npm run dev
```

Сборка для продакшена:

```bash
npm run build
npm run preview
```

## Деплой на Timeweb App Platform

В настройках приложения (Настройки → Параметры сборки):

| Параметр | Значение |
|---|---|
| Команда сборки | `npm run build` |
| Директория сборки | `dist` |

Важно: у Vite результат сборки лежит в `dist`, а не в `build` (как у Create React App). Если указать неверную папку, сервер отдаст исходный `index.html` с `/src/main.tsx` — появится белая страница и ошибка MIME type в консоли.

После деплоя в DevTools → Network проверьте, что загружается `./assets/index-*.js`, а не `/src/main.tsx`.

## Структура

- `src/app/App.tsx` — все секции сайта
- `src/styles/` — шрифты, Tailwind, дизайн-токены
- `public/photos/` — фотографии (копии из папки `Photo`)
- `CV/` — исходное резюме
